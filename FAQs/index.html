OpenCravat is a powerful genomic analysis tool developed by Johns Hopkins University that provides comprehensive annotation of genetic variants found in your VCF file:

  • Multiple Databases: Cross-references variants with numerous scientific databases to provide the most up-to-date information
  • Advanced Visualization: Creates intuitive visualizations that help you understand complex genetic information

The Web3 VCF Annotator currently accepts the following file formats:

  • VCF Files (.vcf): The standard Variant Call Format files produced by most genetic sequencing platforms, including whole genome sequencing and exome sequencing
  • 23andMe Text Files (.txt): The raw data files that you can download from your 23andMe account

For optimal analysis results, we recommend using VCF files whenever possible, as they contain more comprehensive genetic information and allow for more detailed annotation. However, our system is designed to work effectively with both formats.

Important notes about file handling:

  • Compressed files (.zip, .gz) should be extracted before uploading
  • Very large files (>100MB) may take longer to process
  • Files are securely handled and encrypted throughout the analysis process

If you have questions about your specific file format, please contact our support team at support@genobank.io.
